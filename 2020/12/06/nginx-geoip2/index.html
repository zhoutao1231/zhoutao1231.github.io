Nginx 配合 GeoIP模块 限制区域

发表于 2020-12-06 更新于 2020-12-09 分类于 books ， Nginx 阅读次数： Valine：
本文字数： 10k 阅读时长 ≈ 9 分钟

前言

在某些情况下我们不希望某个国家的用户访问我们的网站，这时我们可以使用 Nginx 和 GeoIPLite2 来实现对国家或者区域用户进行访问控制，本文介绍如何使用GeoIP模块让nginx实现限制某个地区用户访问的功能。

  • 注意： MaxMind Geolite 旧版数据库已经停止更新，现在我们应该改用MaxMind GeoIP2或者Geolite2数据和 Nginx_Http_Geoip2 模块。
  • 地理位置限制对于 HTTP 和 TCP/UDP 协议都有效。

一、先决条件

  • 准备 nginx_http_geoip2_module
  • 注册 MaxMind 账号以获取 GeoIP2 或者 Geolite2 数据库
  • Mdblookup程序，用于在MaxMind数据库文件中查找IP地址（可选）
